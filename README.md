# 💱 Vertex FX
[![Live Demo](https://img.shields.io/badge/🚀%20Live%20Demo-00C853?style=for-the-badge)](https://zahilhassan.github.io/Vertex-FX/)
A modern and responsive **Live Currency Converter** built with **HTML, CSS, and JavaScript**.

Vertex FX provides real-time currency conversion using a live exchange rate API with a premium fintech-inspired interface.

---

## ✨ Features

- 🌍 Live Exchange Rates
- 💵 Multi-Currency Converter
- 🔄 Currency Swap
- 📈 Live Currency Ticker
- 🌙 Dark & Light Mode
- 📱 Fully Responsive Design
- 📋 Copy Conversion Result
- 🕒 Conversion History (Local Storage)
- ⚡ Fast & Lightweight
- 🎨 Modern Glassmorphism UI

---

## 🖥️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Live Exchange Rate API
- Local Storage
